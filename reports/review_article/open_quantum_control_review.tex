\begin{abstract}
We present a reproducible review and benchmarking layer for the Open Quantum
Control laboratory, a Python-first environment designed to identify and control
dissipative dynamics in open quantum systems.  The current experimental anchor
is quadrupolar Na-23 nuclear magnetic resonance (NMR), modeled as a spin-$3/2$
platform with tomography, finite-pulse control, dynamical-decoupling (DD)
filtering, and open-system identification.  The manuscript consolidates the
paper reproductions completed so far, the internal physical benchmarks, the
mathematical model stack, the computational-laboratory architecture, and the
experimental-decision workflow prepared for future raw laboratory data.  The
central claim is methodological rather than experimental: every result is tied
to scripts, configuration files, figures, metrics, hashes, and a research-memory
record.  At the present stage, the repository provides a validated synthetic
benchmark layer that sharpens model choice, exposes failure modes, and specifies
which measurements are required before stronger physical claims can be made.
\end{abstract}

\noindent\textbf{Keywords:} open quantum systems; quadrupolar NMR; sodium-23;
spin-$3/2$ dynamics; quantum state tomography; dynamical decoupling; quantum
control; process characterization.

\section{Introduction and Scientific Scope}

The central scientific question addressed here is:
\begin{quote}
\emph{Control and identification of dissipative dynamics in open quantum systems:
a statistical-physics approach with experimental validation.}
\end{quote}

In operational terms, the project asks how one can infer the effective dynamics
of a quantum system interacting with an uncontrolled environment and then use
that inference to select better controls.  In the current repository, the
starting experimental platform is Na-23 NMR, where a single quadrupolar nucleus
with spin $I=3/2$ spans a four-dimensional Hilbert space.  This system is rich
enough to contain an encoded two-qubit structure, quadrupolar splittings,
coherence-order dynamics, state tomography, pseudo-pure state preparation, and
control errors during finite pulses
\citep{spin32_algebraic_2004,qst_relaxation_2008,spin32_qlogic_qst_2005,
quadrupolar_qip_2012,grape_nmr_control_2005,pps_optimal_control_2012}.  The
hardware-facing extensions of the repository are guided by DD filtering,
noise-spectroscopy, multipass QPT, GST, and process-tensor ideas
\citep{noise_filtering_control_2014,dd_noise_spectroscopy_2011,
flux_qubit_noise_spectroscopy_2011,multipass_qpt_2024,
projected_ls_qpt_2022,gate_set_tomography_2021,
nonmarkov_process_tensor_2020,nonmarkov_noise_2022}.

The immediate laboratory problem is not to claim a final physical model.  It is
to build a reproducible framework with the following properties:
\begin{enumerate}
  \item validated spin-$3/2$ operators, Hamiltonians, spectra, and tomography;
  \item paper-level synthetic reproductions that isolate the useful physical
  mechanism of each reference;
  \item quantitative metrics that can be compared across model classes;
  \item a memory layer that records every reproducible result and artifact;
  \item a decision pipeline that can ingest future laboratory data.
\end{enumerate}

\section{Methods: Physical Model and Inference Stack}

\subsection{Spin-$3/2$ Hilbert space}

For a quadrupolar nucleus with spin $I=3/2$, the Hilbert-space dimension is
\begin{equation}
  d = 2I + 1 = 4.
\end{equation}
The computational basis is the Zeeman basis
\begin{equation}
  \left\{ |3/2\rangle, |1/2\rangle, |-1/2\rangle, |-3/2\rangle \right\}.
\end{equation}
The spin operators are generated from the ladder operators,
\begin{align}
  I_x &= \frac{1}{2}(I_+ + I_-),\\
  I_y &= \frac{1}{2i}(I_+ - I_-),\\
  I_z |m\rangle &= m |m\rangle.
\end{align}
The internal validation layer checks Hermiticity, traces, commutators such as
\begin{equation}
  [I_x,I_y] = i I_z,
\end{equation}
and the spectrum of $I_z$.  This matters because all later Hamiltonian, QST, and
control models depend on these operators.

\subsection{Quadrupolar NMR Hamiltonian}

The current effective rotating-frame Hamiltonian is built from quadrupolar,
radio-frequency (RF), offset, and Bloch-Siegert terms:
\begin{equation}
  H(t) =
  H_{Q}^{(1)} + H_{Q}^{(2)} + H_{\mathrm{off}}
  + H_{\mathrm{rf}}(t) + H_{\mathrm{BS}}(t).
\end{equation}
The first-order quadrupolar contribution is modeled as
\begin{equation}
  H_{Q}^{(1)} \propto 3I_z^2 - I(I+1)\mathbb{I},
\end{equation}
which produces a central transition and two satellite transitions.  The current
Na-23 configuration uses $\nu_Q \approx 15625~\mathrm{Hz}$, matching the
synthetic and reference spectra in which the satellites appear near
$\pm\nu_Q$ \citep{na23_relaxometry_2023,spin32_algebraic_2004,
spin32_qlogic_qst_2005,quadrupolar_qip_2012}.

The RF pulse Hamiltonian is
\begin{equation}
  H_{\mathrm{rf}} =
  \omega_1(\cos\phi\,I_x+\sin\phi\,I_y),
\end{equation}
where $\omega_1$ is the RF amplitude and $\phi$ is the transmitter phase.  Finite
pulses are important: the Paper D benchmark showed that ignoring internal
quadrupolar evolution during a long selective pulse can be catastrophically
wrong, whereas GRAPE-style control can recover high fidelity under modeled
detuning and RF-scale errors \citep{spin32_qlogic_qst_2005,grape_nmr_control_2005}.

\subsection{Density matrices, tomography, and open dynamics}

The state is represented by a density matrix $\rho$, with unit trace,
Hermiticity, and positive semidefinite spectrum:
\begin{equation}
  \rho = \rho^\dagger,\qquad \mathrm{Tr}(\rho)=1,\qquad \rho\succeq0.
\end{equation}
Unitary evolution is
\begin{equation}
  \rho(t) = U(t)\rho(0)U^\dagger(t),
  \qquad U(t)=\exp[-iHt].
\end{equation}
For Markovian open dynamics, the standard Lindblad form is
\begin{equation}
  \frac{d\rho}{dt} =
  -i[H,\rho]
  + \sum_k \left(
    L_k\rho L_k^\dagger
    -\frac{1}{2}\{L_k^\dagger L_k,\rho\}
  \right).
\end{equation}
This is not assumed to be universally sufficient.  The non-Markovian benchmarks
explicitly test when a time-local or fixed-channel approximation fails
\citep{nonmarkov_noise_2022,nonmarkov_process_tensor_2020}.

Tomography is implemented as a linear inverse problem.  For each tomography
pulse $U_k$, selected single-quantum coherences are measured:
\begin{equation}
  b_{\alpha k} =
  \mathrm{Tr}\!\left(M_\alpha U_k\rho U_k^\dagger\right).
\end{equation}
Vectorizing the density matrix gives
\begin{equation}
  A\,\mathrm{vec}(\rho)=b,
\end{equation}
which is solved by least squares, followed by Hermitization, trace
normalization, and optional projection onto the positive semidefinite cone.

\subsection{FID, spectrum, and dynamical-decoupling filters}

The NMR free-induction decay (FID) is computed from
\begin{equation}
  s_n = \mathrm{Tr}(\rho_n D),
\end{equation}
where $D$ is the complex receiver observable.  The frequency-domain spectrum is
obtained by a shifted discrete Fourier transform:
\begin{equation}
  \tilde{s}(\nu)=\mathcal{F}[s(t)].
\end{equation}

For dephasing-noise spectroscopy, a control sequence defines a switching
function $y(t)\in\{+1,-1\}$.  The filter function is
\begin{equation}
  F_i(\omega)=\left|\int_0^T y_i(t)e^{i\omega t}\,dt\right|^2.
\end{equation}
The coherence after sequence $i$ is modeled as
\begin{equation}
  C_i = \exp[-\chi_i],
  \qquad
  \chi_i = \frac{1}{\pi}\int_0^\infty S(\omega)F_i(\omega)\,d\omega.
\end{equation}
Discretization gives the inverse problem
\begin{equation}
  -\log C_i = \sum_j A_{ij} S_j,
  \qquad S_j\ge 0,
\end{equation}
which is solved by non-negative least squares in the current synthetic
spectroscopy layer \citep{noise_filtering_control_2014,
dd_noise_spectroscopy_2011,flux_qubit_noise_spectroscopy_2011}.

\section{Methods: Reproducibility Architecture and Benchmark Protocol}

The repository is organized as a computational laboratory rather than as a loose
collection of notebooks.  The practical roles are:
\begin{itemize}
  \item \texttt{src/oqs\_control}: validated physics, tomography, control,
  noise, and hardware-characterization modules;
  \item \texttt{scripts}: one-command entry points for each reproduction or
  workflow;
  \item \texttt{outputs/repro/<paper\_id>/latest}: paper reproduction artifacts;
  \item \texttt{outputs/workflows/<workflow\_id>/latest}: operational workflows;
  \item \texttt{docs}: theory, reproduction notes, validation plans, and
  laboratory documentation;
  \item \texttt{tests}: physical and numerical regression tests;
  \item \texttt{lab/research\_memory}: generated records, hashes, summaries,
  and artifact indices.
\end{itemize}

Each benchmark or reproduction run generates the artifact tuple
\begin{equation}
  \{\mathrm{script},\mathrm{note},\mathrm{figures},
  \mathrm{metrics.json},\mathrm{config\_used.json},\mathrm{results.json},
  \mathrm{run\_metadata.json}\}.
\end{equation}
At the time of this manuscript build, the test suite reports 90 passing tests and the
research-memory layer contains 17 records: 15 paper reproductions, one internal
validation baseline, and one experimental-decision workflow.

\section{Quantitative Evaluation Metrics}

The benchmark layer compares reproductions and workflows through physically
interpretable metrics rather than through a single aggregate score.  The
definitions below are those used repeatedly throughout the present manuscript.

\subsection{Notation conventions}

To keep the notation uniform across NMR, open-system, and hardware-facing
sections, the following conventions are used throughout.  The time-domain NMR
signal is denoted by $s(t)$, whereas its Fourier-domain representation is
$\tilde{s}(\nu)$.  The environmental noise power spectral density is denoted by
$S(\omega)$.  Estimated or reconstructed quantities carry a hat, for example
$\hat{\rho}$ or $\hat{S}(\omega)$, while sample means are denoted by an
overbar.  When two competing protocols are compared, the label ``baseline''
refers to the simpler or previously used procedure, and ``method'' denotes the
candidate replacement.

\subsection{State, gate, and observable metrics}

For reconstructed or propagated density matrices $\rho$ and $\sigma$, the state
fidelity is
\begin{equation}
  F(\rho,\sigma)=
  \left[
    \mathrm{Tr}
    \left(
      \sqrt{\sqrt{\rho}\,\sigma\,\sqrt{\rho}}
    \right)
  \right]^2.
\end{equation}
For unitary control targets $U_{\mathrm{tar}}$ and implemented propagators
$U$, the normalized operator overlap used in the control benchmarks is
\begin{equation}
  F_{\mathrm{op}}(U,U_{\mathrm{tar}})
  =
  \frac{\left|\mathrm{Tr}\!\left(U_{\mathrm{tar}}^\dagger U\right)\right|}{d}.
\end{equation}
For fitted or predicted observables $x_j$ and references $\hat{x}_j$, the
root-mean-square error is
\begin{equation}
  \mathrm{RMSE}(x,\hat{x})
  =
  \sqrt{\frac{1}{N}\sum_{j=1}^{N}(x_j-\hat{x}_j)^2}.
\end{equation}

\subsection{Spectral metrics}

For reconstructed and reference noise spectra sampled on the same grid, the
reported relative spectrum error is
\begin{equation}
  \epsilon_{S}
  =
  \frac{\left\|S-\hat{S}\right\|_{2}}{\left\|\hat{S}\right\|_{2}}.
\end{equation}
The spectrum-correlation metric used in the DD spectroscopy and decision
pipeline layers is the Pearson coefficient
\begin{equation}
  r_{S}
  =
  \frac{\sum_j (S_j-\bar{S})(\hat{S}_j-\bar{\hat{S}})}
  {\sqrt{\sum_j (S_j-\bar{S})^2}\sqrt{\sum_j (\hat{S}_j-\bar{\hat{S}})^2}}.
\end{equation}
These two quantities are intentionally reported together: correlation captures
shape agreement, whereas $\epsilon_S$ captures amplitude-level mismatch.

\subsection{Improvement and memory diagnostics}

When two methods are compared against the same target observable, the reported
improvement factor is
\begin{equation}
  \mathcal{I}
  =
  \frac{E_{\mathrm{baseline}}}{E_{\mathrm{method}}},
\end{equation}
where $E$ denotes an error metric such as RMSE or process-reconstruction error.
For coherence-preservation comparisons, a gain factor
\begin{equation}
  \mathcal{G}
  =
  \frac{C_{\mathrm{method}}}{C_{\mathrm{baseline}}}
\end{equation}
is used instead.  For non-Markovian diagnostics, the benchmark layer uses the
Breuer--Laine--Piilo backflow functional
\begin{equation}
  \mathcal{N}_{\mathrm{BLP}}
  =
  \int_{\dot{D}(t)>0}\frac{d}{dt}D\!\left(\rho_1(t),\rho_2(t)\right)\,dt,
\end{equation}
where $D$ is the trace distance between two evolved states.  No single metric is
treated as sufficient in isolation; each numerical value is interpreted in the
context of the physical observable and of the associated reference paper.

\section{Results: Scientific Synthesis across the Reproduced Literature}

\subsection{Quadrupolar-NMR modeling, tomography, and control}

The Na-23 and spin-$3/2$ references establish a coherent modeling chain from
operator algebra to state preparation, tomography, and pulse design
\citep{na23_relaxometry_2023,spin32_algebraic_2004,qst_relaxation_2008,
spin32_qlogic_qst_2005,quadrupolar_qip_2012,grape_nmr_control_2005,
pps_optimal_control_2012}.  Three quantitative conclusions are especially
important.  First, the algebraic layer is numerically stable: Hilbert-space and
Liouville-space propagation agree to within $2.55\times10^{-17}$ in the FID
benchmark.  Second, density-matrix-level identification is substantially more
informative than spectrum-only fitting: the tomography workflow recovers the
synthetic relaxation rates exactly in the noiseless limit, with mean QST
fidelity equal to one.  Third, finite-pulse modeling is indispensable:
rectangular selective control can fail catastrophically once quadrupolar
evolution during the pulse is retained, whereas GRAPE optimization restores a
mean robustness-grid fidelity of $0.962$.

Within this same group, the relaxometry and pseudo-pure-state benchmarks clarify
how the laboratory should interpret future data.  The relaxometry layer shows
that biexponential envelopes remain useful as empirical descriptions, but that
microscopic interpretation requires spectral-density parameters rather than
envelope fitting alone.  The pseudo-pure-state layer shows that encoded control
experiments can be initialized reproducibly, with analytical and optimized
preparation procedures reaching near-unit deviation-state fidelity in the
synthetic setting.

\subsection{Noise spectroscopy, memory diagnostics, and control selection}

The noise and memory reproductions define the model-selection logic of the
laboratory \citep{nonmarkov_noise_2022,nonmarkov_process_tensor_2020,
noise_filtering_control_2014,dd_noise_spectroscopy_2011,
flux_qubit_noise_spectroscopy_2011}.  The Markovian benchmark produces a BLP
backflow value of $5.457$, a negative-rate fraction of $0.483$, and a
Markovian-fit RMSE of $0.273$, demonstrating that a simple Lindblad description
cannot be accepted merely because it is convenient.  Once the model class is
restricted to dephasing-noise filtering, DD control yields a clear operational
gain: the best tested CPMG sequence raises synthetic coherence from $0.609$ to
$0.998$.  The spectroscopy layer then turns coherence data into inference: the
reconstructed colored spectrum reaches correlation $0.923$ with the synthetic
truth and localizes a narrow spectral feature to within the discretization
resolution.

The flux-qubit benchmark is important because it shows that the spectroscopy
logic is not specific to Na-23 NMR.  In the hardware-facing synthetic test, the
power-law fit returns $\alpha=0.7399$ for a true $1/f^{0.78}$ spectrum, which
supports carrying the same inference machinery toward gate-model quantum
hardware once calibrated coherence data become available.

\subsection{Process characterization for hardware-facing validation}

The tomography and characterization reproductions define the route from
open-system inference to hardware validation
\citep{multipass_qpt_2024,projected_ls_qpt_2022,gate_set_tomography_2021}.
Multipass QPT reduces the synthetic PTM reconstruction error from $0.216$ to
$0.0342$ at the best pass number, an improvement factor of $6.30$.  Projected
least-squares QPT shows that physical constraints are not optional: projection
onto the CPTP set reduces the mean Choi error from $0.245$ to $0.177$ while
removing unphysical negative-eigenvalue violations.  GST then addresses the
remaining ambiguity due to SPAM, improving held-out prediction RMSE from
$0.0655$ to $0.00762$ under self-consistent fitting.  Taken together, these
results imply that future hardware experiments should be interpreted through a
hierarchy: ordinary tomography when assumptions are controlled, physical
projection when finite-data artifacts dominate, and GST-style predictive fitting
when SPAM or gauge issues become the limiting factor.

\section{Results: Integrated Experimental-Decision Workflow}

The new workflow composes the validated modules into a single decision loop:
\begin{enumerate}
  \item prepare a reference spin-$3/2$ state;
  \item validate preparation/readout with synthetic QST;
  \item acquire or simulate DD coherence measurements;
  \item reconstruct an effective dephasing spectrum;
  \item score candidate control sequences;
  \item select the sequence with highest predicted coherence;
  \item compare predictions against real laboratory measurements when available.
\end{enumerate}
This operational layer combines the tomography-identification logic of
\citep{qst_relaxation_2008}, the finite-pulse warning from
\citep{spin32_qlogic_qst_2005}, the filter-function viewpoint of
\citep{noise_filtering_control_2014}, and the DD spectral inversion strategy of
\citep{dd_noise_spectroscopy_2011}.

In the current synthetic run, the state-preparation QST fidelity is
$0.9856$, the reconstructed spectrum has correlation $0.9326$, and the selected
control sequence is CPMG-24 with predicted coherence $0.9895$.  The lab
comparison status is \texttt{waiting\_for\_lab\_data}.

\section{Discussion}

\subsection{Scope of the present claims}

The present benchmark layer supports the following claims:
\begin{itemize}
  \item the Python spin-$3/2$ operator and Hamiltonian layer is internally
  validated by tests;
  \item the Na-23 FID and spectrum pipeline reproduces the expected central and
  satellite transition structure in the synthetic/reference setting;
  \item QST can reconstruct synthetic spin-$3/2$ density matrices and estimate
  relaxation rates under controlled assumptions;
  \item finite-pulse dynamics cannot be ignored for selective quadrupolar-NMR
  gates;
  \item optimal-control pulses are necessary for robust coherent operations in
  the encoded spin-$3/2$ manifold;
  \item DD filtering and DD spectroscopy provide a concrete route from
  coherence data to an effective noise spectrum;
  \item Markovian, GST, QPT, multipass-QPT, and process-tensor layers now exist
  as synthetic benchmarks for future hardware-facing characterization.
\end{itemize}

The present study does not yet support the following stronger claims:
\begin{itemize}
  \item that a real Na-23 sample follows the current synthetic relaxation
  model;
  \item that the selected CPMG-24 sequence is experimentally optimal;
  \item that a Lindblad model is sufficient for the real laboratory data;
  \item that hardware-level SPAM, pulse calibration, or readout errors have
  already been characterized.
\end{itemize}

\subsection{Model limitations and escalation criteria}

Disagreement with incoming data should be treated as physical information rather
than as an implementation defect.  The escalation logic is:
\begin{enumerate}
  \item If FID and spectrum peaks are shifted, inspect $\nu_Q$, offset, dwell
  time, phase convention, and apodization.
  \item If tomography residuals are large but spectra look reasonable, inspect
  pulse calibration, receiver phase, QST linear conditioning, and PSD projection.
  \item If DD coherences cannot be fitted by a positive spectrum, inspect pulse
  errors, relaxation channels beyond pure dephasing, and measurement
  uncertainty.
  \item If echo recovery or multi-time correlations disagree with a fixed
  Markovian model, escalate to memory-effective models or process tensors.
  \item If process reconstructions are non-physical, project onto the CPTP set
  or use constrained estimators.
  \item If gate errors are inseparable from SPAM, use GST-style predictive
  fitting rather than ordinary QPT alone.
\end{enumerate}

\section{Experimental Validation Protocol}

When new lab data arrive, each run should provide:
\begin{itemize}
  \item raw data path and immutable copy;
  \item experiment identifier, date, operator, sample/device identifier;
  \item pulse sequence definitions and calibration values;
  \item dwell time, scans/shots, receiver phase, and preprocessing convention;
  \item uncertainty estimate for every coherence, spectrum, or tomography datum;
  \item comparison manifest filled from
  \texttt{lab\_manifest\_template.json};
  \item model snapshot and configuration hash.
\end{itemize}

The operational question for the first laboratory campaign is:
\begin{quote}
Does the current synthetic-to-effective model predict the measured coherences
and QST observables within uncertainty?  If yes, use the selected control
sequence.  If no, identify which assumption failed and escalate the model
class.
\end{quote}

\section{Conclusions}

The present manuscript converts an initial MATLAB-based Na-23 NMR effort into a
Python-first computational research laboratory.  The repository now contains a
validated spin-$3/2$ model, a reproducible suite of paper-grounded benchmarks,
a research-memory layer, and an experimental-decision workflow.  The scientific
value is structural: each claim is attached to code, configuration, metrics,
figures, and tests, and each synthetic success is explicitly separated from the
stronger claims that still require experimental validation.  That structure is
the prerequisite for turning future NMR and quantum-hardware measurements into
defensible physical conclusions.
