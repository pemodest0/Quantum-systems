# Paper Card: Manzano 2013

## Source

Manzano, D. "Quantum Transport in Networks and Photosynthetic Complexes at the Steady State." PLOS ONE 8, e57041 (2013). DOI: https://doi.org/10.1371/journal.pone.0057041

## Physical question

How does dephasing affect transport when excitation is continuously injected and removed, instead of studying only one initially prepared excitation?

## Model

Open quantum network at steady state, with source, drain/output, coherent couplings, and dephasing.

## Central equations

```text
0 = -i[H, rho_ss] + L_source(rho_ss) + L_drain(rho_ss) + L_dephasing(rho_ss)
```

## Observables

- steady-state current.
- dephasing-dependent transport.
- network-dependent response.

## Main result

Noise-enhanced transport can appear in steady-state scenarios, but the interpretation differs from finite-time target arrival.

## Limitations

Steady-state current is not the same observable as single-excitation arrival probability. The two should not be mixed without care.
