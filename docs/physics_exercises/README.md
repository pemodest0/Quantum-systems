# Physics Exercise Bank

The local assistant uses this exercise bank for retrieval and automatic evaluation. Each exercise has a short answer, a common mistake, and grading criteria.

Categories:

- `conceptual`: basic meaning of states, density matrices, graphs, dephasing, target arrival, loss, entropy, and disorder.
- `figure_reading`: how to read heatmaps, dashboards, surfaces, animations, and uncertainty.
- `transport_lindblad`: equations and open-system transport reasoning.
- `trap`: conceptual traps where a confident but wrong answer is common.

The bank is not a substitute for a full textbook. It is a controlled local reference for the assistant to avoid unsupported claims.
